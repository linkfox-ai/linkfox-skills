# 1688 — Tool Reference

## DLD Product Billboard (1688 Bestseller Rankings)

**Endpoint**: `POST https://tool-gateway.linkfox.com/dld/productBillboard`
**Auth**: Header `Authorization: <api_key>` (env: LINKFOXAGENT_API_KEY)

### Request Parameters

**Core Parameters:**

| Parameter | Type | Required | Default | Description |
|-----------|------|----------|---------|-------------|
| keyWord | string | No | — | Search keyword (must be Chinese, translate first if not). Max 50 chars |
| date | string | No | — | Query date. Weekly: Sunday date (e.g. `2025-06-15`, last 90 days). Monthly: 1st of month (e.g. `2025-06-01`, last 12 months) |
| pageType | integer | No | 3 | Billboard type: `2`=weekly, `3`=monthly |
| pageIndex | integer | No | 1 | Page number (from 1) |
| pageSize | integer | No | 20 | Results per page (10-100) |

**Sorting:**

| Parameter | Type | Required | Default | Description |
|-----------|------|----------|---------|-------------|
| sortField | string | No | orderCount | Sort field: `orderCount`(orders), `saleCount`(units sold), `saleVolume`(est. revenue), `offerCreateTime`(listing date), `price`(wholesale price), `consignPrice`(dropship price) |
| sortType | string | No | desc | Sort direction: `desc`, `asc` |

**Keyword Match:**

| Parameter | Type | Required | Default | Description |
|-----------|------|----------|---------|-------------|
| searchType | integer | No | 1 | Keyword match: 1=fuzzy, 3=exact |

**Product & Seller Tags:**

| Parameter | Type | Required | Default | Description |
|-----------|------|----------|---------|-------------|
| offerType | integer | No | 0 | Product tag: 0=any, 2=new product, 3=1688 Select, 4=cross-border, 5=customizable, 6=store treasure |
| companyType | integer | No | 0 | Company type: 0=any, 1=store, 2=factory |
| shiLiType | string | No | — | Seller tier (comma-separated): `superFactory`, `Power`, `TrustPass` |
| beginTpYear | integer | No | — | TrustPass years (min) |
| endTpYear | integer | No | — | TrustPass years (max) |

**Price & Volume Filters:**

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| beginPrice | number | No | Wholesale price (min) |
| endPrice | number | No | Wholesale price (max) |
| beginConsignPrice | number | No | Dropship price (min) |
| endConsignPrice | number | No | Dropship price (max) |
| beginOrderCount | integer | No | Order count (min) |
| endOrderCount | integer | No | Order count (max) |
| beginSaleCount | integer | No | Units sold (min) |
| endSaleCount | integer | No | Units sold (max) |
| beginSaleVolume | number | No | Est. revenue (min) |
| endSaleVolume | number | No | Est. revenue (max) |
| beginStartQuantity | integer | No | Min order quantity (min) |
| endStartQuantity | integer | No | Min order quantity (max) |

**Logistics & Service Filters:**

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| sendTime | string | No | Shipping time in hours (comma-separated): `24`, `48`, `72` |
| proxyRights | string | No | Dropship benefits (comma-separated): `4360897`(free shipping dropship), `449154`(buy-first-pay-later) |
| shopService | string | No | Seller services (comma-separated): `4057409`(worry-free purchase), `888777`(deep verification report) |
| buyerProtections | string | No | Buyer protections (comma-separated): `商品包邮`, `7天包退货`, `支持运费险` |
| faceToFaceSupport | string | No | Shipping label support (comma-separated): `441218`(Taobao), `386434`(Douyin), `422914`(Pinduoduo), `422978`(Xiaohongshu), `386370`(Kuaishou) |

**Other:**

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| productIds | string | No | Product IDs (Chinese comma separated, max 20) |
| goodsUrl | string | No | Direct product URL for lookup |
| beginOfferCreateTime | string | No | Listing date start (YYYY-MM-DD) |
| endOfferCreateTime | string | No | Listing date end (YYYY-MM-DD) |

### Response Fields

**Top-level:**

| Field | Type | Description |
|-------|------|-------------|
| total | integer | Record count |
| type | string | Render style |
| columns | array | Render column definitions |
| products | array | Product list |

**Product object:**

| Field | Type | Description |
|-------|------|-------------|
| offerId | string | Product ID |
| asin | string | Product code |
| title | string | Product title |
| price | number | Wholesale price (CNY) |
| consignPrice | number | Dropship price (CNY) |
| currency | string | Currency |
| unit | string | Unit of measure |
| quantityBegin | integer | Min order quantity |
| quantityPrices | string | Volume pricing tiers |
| salesOrderCount | integer | Order count (by period) |
| salesQuantity | integer | Units sold (by period) |
| estimatedSalesAmount | integer | Est. revenue (by period) |
| dataType | string | Data type: `weeklyData` or `monthlyData` |
| availableDate | string | Listing date (yyyy-MM-dd HH:mm:ss) |
| deliveryTime | string | Delivery time |
| levelName | string | Category hierarchy name |
| company | string | Shop name |
| shopId | string | Shop ID |
| shopUrl | string | Shop link |
| asinUrl | string | Product link |
| imageUrl | string | Product image URL |
| sourceType | string | Source platform (1688) |
| sourceTool | string | Source tool |

### Key Usage Notes
- Keywords must be in **Chinese** — translate non-Chinese keywords before querying.
- Weekly billboard date must be a **Sunday**; monthly billboard date must be the **1st of the month**.
- Weekly data: last 90 days only. Monthly data: last 12 months only.
- Page size max: 100. Product ID limit: 20.
- All prices in CNY.

## DLD Product Search (1688 Selection)

**Endpoint**: `POST https://tool-gateway.linkfox.com/dld/productSearch`
**Auth**: Header `Authorization: <api_key>` (env: LINKFOXAGENT_API_KEY)

### Request Parameters

**Core Search:**

| Parameter | Type | Required | Default | Description |
|-----------|------|----------|---------|-------------|
| keyWord | string | No | — | Search keyword (must be Chinese, max 50 chars) |
| goodsUrl | string | No | — | Product URL (alternative to keyword) |
| productIds | string | No | — | Product IDs (comma-separated, max 20) |
| cycle | string | No | — | Period: `7`(last 7 days) or `30`(last 30 days) |
| searchType | integer | No | 1 | Match: 1=fuzzy, 3=exact |

**Sorting:**

| Parameter | Type | Required | Default | Description |
|-----------|------|----------|---------|-------------|
| sortField | string | No | orderCount30d | Sort field: `orderCount7d`, `saleCount7d`, `saleVolume7d`, `orderCount30d`, `saleCount30d`, `saleVolume30d`, `offerCreateTime`, `price`, `consignPrice` |
| sortType | string | No | desc | Sort direction: `desc`, `asc` |

- All prices in CNY.
- Supports both wholesale price (`price`) and dropship price (`consignPrice`) — always show both for margin comparison.
